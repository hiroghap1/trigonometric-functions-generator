🎨 CSS 書き出し

@property + CSS の sin()/cos() で波形と円運動を再現します。 プレビューを確認し、コードをコピーして HTML ファイルに貼り付けてください。

📖 これは何？（仕組みの解説）

@property は CSS のカスタムプロパティに「型」を付けて登録する仕組み。 通常 --x: 0 のような変数は文字列扱いなので CSS アニメーションで 滑らかに補間できませんが、@property で <number> や <angle> として登録すると数値として補間できるようになります。

このアプリが書き出すコードは：

  • 波形ドット：--t を 0→1 に補間し、x 座標は --t に比例、 y 座標は sin(--t × 360deg) に比例。CSS の sin() 関数が直接呼べるので、 JavaScript なしで本物の正弦波運動になります
  • 円運動ドット：--theta を 0deg→360deg に補間し、 位置を (cos(--theta)·R, sin(--theta)·R) に置く。 これも CSS の cos() / sin() が計算してくれます

対応ブラウザ：Chrome 111+ / Safari 15.4+ / Firefox 128+

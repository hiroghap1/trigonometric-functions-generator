# 🌸 三角関数ジェネレーター

「三角関係ジェネレーター」のかわいい操作感そのままに、テーマを **三角関数** に置き換えた学習・可視化ツール。
ノードを繋いで関数を組み立て、相関図のように整理しながら、波形・音・式・グラフを同時に確かめられます。

- インストール不要 / サーバー送信なし / すべてブラウザ内で完結
- ローカル `index.html` をブラウザで開くだけで動作
- 編集内容は localStorage に自動バックアップ

---

## 🚀 使い方

```bash
open index.html
# または
python3 -m http.server  # → http://localhost:8000
```

1. 右サイドバー or 上部「＋ノード」から関数を追加
2. ノード右の「＋」をドラッグして別ノードへ接続
3. 📈 出力ノードに繋ぐと波形・式が即時プレビュー
4. 線をクリックすると関係ラベル / 矢印スタイル変更
5. 📸 スクショ・💾 保存・🧮 LaTeX で持ち出し

## ✨ 機能一覧

### 関数ノード
- 🌸 sin / 🍀 cos / 🔥 tan / ❄️ cot / ☀️ sec / 🌙 csc
- 🍡 定数（任意の値）
- ⭕ 単位円（角度θスライダー、補助線・三角値表示、自動回転）

### 演算 / 変換
- ➕ 合計 / ➖ 差 / ✖️ 積 / ➗ 商
- 🎀 平行移動（dx, dy） / 🎁 拡大縮小（sx, sy） / 🪞 反転（x軸 / y軸）
- ⚡ 微分（中央差分） / ★ 積分（台形則）

### 出力モード
- グラフ `y = f(x)`
- リサジュー（入力2つを `(x(t), y(t))` で描画）
- 極座標 `r = f(θ)`
- **重ね描き**: 出力に複数入力を繋ぐと別の色で同時にプロット
- **値ツールチップ**: グラフ上にマウスを置くと `x, y1, y2, …` をリアルタイム表示

### 関係線
- 関係ラベル: ♡好き / ✕嫌い / ⚡微分 / ★積分 / ？気になる / 自由テキスト
- 矢印スタイル: 片方向 / 両方向 / 点線 / 波線
- 線をクリック → クイック編集ポップアップ
- 中点ハンドルをドラッグして自由カーブ調整（PC）

### キャンバス操作
- 空き地ドラッグでパン
- マウスホイールでズーム（カーソル中心、ゆるやか連続変化）
- 右下の `＋ / − / ⌂` ズームコントロール、ズーム率表示

### 複数選択 / コピペ
- Shift+クリック / Ctrl+クリックで追加選択
- Shift+空き地ドラッグで矩形選択
- Ctrl+A 全選択
- Ctrl+C / V / D でコピー & ペースト & 複製
- 複数選択したまま一括ドラッグ・一括削除

### PC限定レイアウト機能
- 🌈 グループ（複数所属可、色変更、付箋風囲み）
- 📝 メモ（textarea / 色 / 角度 / リサイズ / ノードに点線で接続）

### 数式パーサ入力
- 例: `sin(2x) + cos(x)` / `2sin(x) - 3cos(2x+1)` / `sin(x)*cos(x)`
- 対応: sin / cos / tan / cot / sec / csc, π, +, −, *, /, 暗黙の積（`2x`, `2sin`）, 括弧
- 関数引数は `Bx + C` の形に自動分解。係数 `2sin(x)` などは `a` パラメータに畳み込み
- 生成後は自動レイアウト

### 自動整列
- 出力ノードを起点に、依存深さに応じて左→右へ階層配置
- 出力ノードがない場合は左から右への単純整列

### 履歴 & 保存
- Ctrl+Z / Ctrl+Y / Ctrl+Shift+Z（最大50件）
- localStorage 自動バックアップ
- `.sankakukansu.json` 書き出し / 読込（旧形式は自動 migrate）

### 出力 / 共有
- 📸 スクショ（html2canvas で PNG、CDN失敗時は SVG fallback）
- 🧮 LaTeX をクリップボードへコピー
- 🔊 出力波形を Web Audio API で **音として再生**（220Hz基準・1.5秒）
- ▶ アニメーション再生（`useTime` ノード・単位円の自動回転と連動）

### クイズ / 学習
- 🎯 **練習モード**: 出題されたグラフを再現するとお祝い演出。15問のステップアップ式（sin(x) → 倍音合成、ピタゴラス恒等式まで）。ヒント・スキップあり、自動で次の問題へ
- 🎲 クイズモード: ランダム式のグラフから 3択で正解を当てる
- 📘 学習モード: ノード hover で解説ツールチップ
- 💡 三角関数 Tips パネル: 20種以上の公式・現象（ピタゴラスの恒等式 / 倍角・和積 / フーリエ / リサジュー / オイラー / AM・FM変調 / カージオイド など）
- 🌍 **どこで使われてる？** パネル: 25種の身近な実例（音楽・GPS・MRI・ゲームのキャラ回転・スカイツリーの揺れ対策・コンセントの電気・Siri音声認識・宇宙開発・虹・倍音と音色 など）
